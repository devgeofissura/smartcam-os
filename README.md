# smartcam-os
O SmartCam OS é uma plataforma embarcada desenvolvida para a T-SIMCAM ESP32-S3 com o objetivo de fornecer uma infraestrutura única para aplicações de:  Rastreamento de pessoas Rastreamento facial Controle de motores Automação Visão computacional Monitoramento industrial Monitoramento de fissuras (Geofissura) Processamento de imagens
